The Refersion for WooCommerce plugin allows shop owners to connect their WooCommerce shop with a Refersion account.

== Description ==
Imagine generating more sales through commission based partnerships with Bloggers, Internet Marketers, Publishers, and so forth. Using Refersion, you\'ll be able to track performance for all of your Affiliates and generate more streams of revenue at the same time.

It only takes a few minutes to setup your Affiliate Program and there is zero technical knowledge required. Just take a look at our Video Quick Start Guide.

Here are some additional features that you can expect in Refersion:

* Complete end-to-end tracking
* We support PayPal!
* Recruit Affiliates
* Swift support

== Installation ==
1. Upload the `refersion-for-woocommerce` directory into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
